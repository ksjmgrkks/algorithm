### 문제 설명

<p>Given an integer array nums of 2n integers,</p>
<p>group these integers into n pairs (a1, b1), (a2, b2), ..., (an, bn) such that the sum of min(ai, bi) for all i is maximized.</p>
<p>Return the maximized sum.</p>
